Google Summer of Code 2012

Welcome to the site for tracking my progress during the Google Summer of Code. I'm working with SilverStripe to refactor the SilverStripe Framework's module system. This will include integrating with Composer. The aim is to make it easier for both module maintainers and users to create, share and use modules, as well as establish a functional API around the module system. I'll try to keep this site updated with my plans and progress over the course of the project.
